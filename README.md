# 🛠️ Teste - Roblox Studio

Este repositório contém uma série de testes e scripts desenvolvidos envolvendo as mecânicas do **Roblox Studio**. Os testes desenvolvidos envolvem manipulação de cenário, interações de jogadores, serviços nativos (Services) e a física do motor do jogo.

---

## 🚀 Como utilizar este repositório

1. Faça o download ou clone este repositório para o seu computador.
2. Abra o arquivo do mapa (`.rbxl`, caso disponível) diretamente no **Roblox Studio**.
